# Local Block Oracle Testnet

## Prerequisites

- [Docker](https://docs.docker.com/get-docker/)
- [Overmind](https://github.com/DarthSim/overmind)
- curl, jq, psql, sh
- JS/TS stuff: yarn, typescript


## Run

`overmind s`

## Useful commands

- Load env file in shell:

  ```sh
  . .overmind.env
  ```
